A / AN / SOME
A
(Singular + Consonant)		 I would like a banana.
He has a sandwich.
AN
(Singular + Vowel)		 Do you want an apple?
I want an egg.
SOME
(Plural / Uncountable)		 I'd like some grapes. (Plural)
I need some milk. (Uncountable)
SOME / ANY
(+) Affirmative Use SOME
I have some bread.
(-) Negative Use ANY
I don't have any coffee.
(?) Questions
(General)		 Use ANY
Do you have any sugar?
(?) Questions
(Offers / Requests)		 Use SOME
Would you like some tea?
