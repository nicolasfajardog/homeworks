📌 Grammar Notes

1. Regular Plurals in English:

  • Add -s: carrot → carrots / onion → onions
  • Add -es (ends in o, ch, sh, s, x): tomato → tomatoes / potato → potatoes
  • Change y → ies (consonant + y): cherry → cherries
  • Change f/fe → ves: half → halves / loaf → loaves

2. Preferences & Offers:
"I prefer chicken rather than beef."
"Do you like broccoli?" → "Yes, I do" / "No, I don't."
"Do you like ribs?" → "Yes, I like them" / "No, I don't like them."
"Would you like some coffee?" → "Yes, please." / "No, thank you."
